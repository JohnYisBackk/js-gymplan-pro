## About

GymPlan PRO is a personal workout tracker built for my own training.

As someone who trains regularly, I wanted a simple application that allows me to track workouts, monitor progress, save personal records and organize training cycles exactly the way I need.

This project was created primarily for my own daily use in the gym and later evolved into a complete workout tracking application.

---

## Features

✅ Workout management

✅ 12-week training cycle system

✅ Personal Records (PR tracking)

✅ Exercise progress tracking

✅ Workout timer

✅ Workout history

✅ Fun Set notes

✅ Exercise descriptions

✅ Weekly progression system

✅ Light / Dark mode

✅ Custom workout images

✅ Mobile optimized design

✅ Data export & import

✅ Local storage persistence

## Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## Why I Built This

Most workout apps felt either too complicated or required subscriptions.

Since I train regularly, I decided to build my own solution tailored specifically to my training style and progression system.

GymPlan PRO is the application I personally use during my workouts.
